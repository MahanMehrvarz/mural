# MURAL Design Context

## Design Context

### Users
Makers, artists, and institutions who own or build the MURAL wall plotter. Technical enough to work with SVG files and web tools. Use the device in workshops, studios, or gallery settings. MURAL Studio is used at a desk before heading to the wall — it's a preparation tool, not a realtime control surface.

### Brand Personality
Precise · Artisanal · Unhurried

The interface should feel like a well-made physical tool — confident and purposeful without being corporate or flashy. It makes large-scale art; it should feel worthy of that.

### Aesthetic Direction
**Atelier Precision** — the feel of a professional print studio or architect's drafting table.

- **Background**: warm cream (`oklch(96.5% 0.01 72)`) — like quality paper
- **Surface**: near-white cards with a single-pixel warm border
- **Accent**: terracotta/brick (`oklch(52% 0.14 35)`) — ink-like, warm, not corporate
- **Success**: muted sage green
- **Typography**: Space Grotesk (headings) + DM Sans (body) — geometric but human
- **Step numbers**: large, dimmed, used as visual anchors in the left column
- **Theme**: Light only — warm, not sterile

Anti-references: generic SaaS blue, dark hacker terminal, glassmorphism, gradient text.

### Design Principles
1. **Function first, ornament second** — every visual element should have a purpose
2. **Warm neutrals over cold grays** — tint everything toward the paper/ink palette
3. **Numbered steps as anchors** — the workflow is linear; make the sequence visible
4. **Left-aligned, unhurried** — no centering everything; give content room to breathe
5. **Restraint in motion** — only animate state changes (progress, done); nothing decorative
