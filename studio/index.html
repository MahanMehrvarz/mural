MURAL Studio

Prepare artwork offline — process any SVG and export a commands file for the plotter.

01
Step one
Upload SVG

Choose your artwork file. SVG line art works best — black strokes on white. Filled shapes will be traced as outlines.

Drop an SVG here, or click to browse
02
Step two
Pin-to-pin Distance

Measure the distance between the two anchor pins on your wall. This sets the drawing width in mm. Height is computed automatically from the artwork's proportions.

— Enter pin-to-pin distance, then upload an SVG

Enter pin-to-pin distance to set the drawing scale

03
Step three
Position & Zoom

Move and scale the artwork within the canvas. The canvas boundary stays fixed — only the artwork shifts. Panning down expands the canvas downward to include everything. Use Reset to start over.

04
Step four
Processing Options

Choose how to convert the artwork into pen paths. You can re-process with different settings before exporting — it doesn't affect the uploaded file.

Render mode

Follows SVG paths directly. Fast and accurate. Best for line art.

Rasterises first, then re-traces. Preserves stroke widths but takes longer.

Infill density 0

0 = outlines only. Higher values fill closed shapes with horizontal lines.

Despeckle 2

Removes small noise blobs from the rasterised image.

Merges overlapping filled areas so the pen doesn't retrace the same region twice.

05
Step five
Process & Export

Processing runs entirely in your browser — no internet needed. It may take 10–60 seconds for complex artwork. Review the path preview before exporting.
